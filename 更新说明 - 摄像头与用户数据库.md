## 📋 使用说明

### 第一次使用

1. **启动 Web 平台**
   ```bash
   npm run dev
   ```

2. **注册账户**
   - 访问 `http://localhost:5173`
   - 点击"去注册"
   - 输入用户名、邮箱、密码
   - 点击"立即注册"
   - ✅ 用户数据已保存到 localStorage

3. **登录**
   - 输入用户名和密码
   - 点击"登录系统"
   - ✅ 登录后会保存到 localStorage

4. **刷新页面测试**
   - 按 `F5` 刷新
   - ✅ 自动登录，无需重新输入密码

---

### 配置小车视频

1. **在小车上运行脚本**
   ```bash
   ./run
   ```

2. **在 Web 平台配置 IP**
   - 进入驾驶舱页面
   - 点击右上角"小车 IP"
   - 输入：`192.168.0.5`
   - 保存并重启

3. **验证视频**
   - 监控页面中间应显示小车实时画面
   - 控制页面中间也应显示小车实时画面
   - 右上角显示 FPS（15-30）

